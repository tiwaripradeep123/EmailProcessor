How to convert PDF to text in .NET

Libraries

You need to copy the following DLLs to the lib directory

lib/reference

  • IKVM.OpenJDK.Core.dll
  • IKVM.OpenJDK.SwingAWT.dll
  • pdfbox-1.8.9.dll

lib/copy

  • commons-logging.dll
  • fontbox-1.8.9.dll
  • IKVM.OpenJDK.Text.dll
  • IKVM.OpenJDK.Util.dll
  • IKVM.Runtime.dll

Download

These PDFBox DLLs can be downloaded here:

  • http://www.squarepdf.net/pdfbox-in-net-download/
